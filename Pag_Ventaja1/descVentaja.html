Configuración de GitBook para GitHub Copilot

  1. Instalar la aplicación

    2. En primer lugar, encontrar la extensión GitBook para GitHub Copilot en el mercado de GitHub e instalarla en las cuentas u organizaciones que desee.

  2. Configura la integración de GitBook

    3. Ve a GitBook e instale la integración de GitHub Copilot en la organizacion/es que quiera.

  3. Autentica GitHub

    4. Inicie sesión en su cuenta de GitHub y seleccione la organización/es para la que desea que funcione la integración.

  4. Conecte sus documentos

    5. En la configuración de integración de GitBook, agregue todos los espacios que desee en GitBook. Los espacios que añadas se utilizarán para proporcionar conocimientos y respuestas cuando hagas una pregunta en GitHub o VS Code.

    Una vez hecho eso, puedes volver a GitHub para probarlo haciendo una pregunta sobre tus documentos.
